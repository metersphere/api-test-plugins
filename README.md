MeterSphere 接口测试插件
=============================

从v3.x版本开始支持扩展覆盖整个接口测试的相关插件，涵盖扩展协议，逻辑控制器，前后置操作等。

### 版本说明

插件的版本要求：兼容 JMeter5.6 版本后的所有插件。

### 下载插件

请在该项目的 [release](https://github.com/metersphere/api-test-plugins/releases) 页面下载对应版本的平台插件。
